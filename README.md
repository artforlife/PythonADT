# PythonADT
Implementation of common and not-so-common Abstract Data Types in Python

Usage:
 
 Using LinkedList
~~~~
from PythonADT.lists import LinkedList
List = LinkedList()
List.push(1)
List.size()
print(List)
~~~~

 Using Stack
~~~~
from PythonADT.stacks import Stack
Stack = Stack()
Stack.push(1)
Stack.size()
Stack.push(3)
Stack.peek()
print(Stack)
Stack.pop()
~~~~
